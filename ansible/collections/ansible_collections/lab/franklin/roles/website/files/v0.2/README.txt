

        <h2>Installing Latest Forge to Server</h2>
        <ul>
            <li>Download minecraft forge JAR to server.</li>
            <li>mv forge-1.15.2-31.2.0-installer.jar forge/</li>>
            <li>java -jar forge-1.15.2-31.2.0-installer.jar --installServer</li>
            <li>Move the new forge jar into the /home/minecraft/forge directory.</li>
            <li>Start the server with the script forge/minecraft.sh in a screen session.</li>
        </ul>
