We have been running a server continuously for about two years now and had fun mining and building many interesting things. Note that this is a survival server, not a creative server. Please don't waste time begging for admins to toggle your gamemode.

Voting

You can earn money in game for things like creating and expanding your residence by casting votes for this server.

  • Cujos Server at Planet Minecraft
  • Cujo's Server at Minestatus.net
  • Vote for us on Topg.org

Server Information

The server runs the latest version of CraftBukkit. That means that you should be able to download and run the latest vanilla minecraft client without any extra effort on your part. If you are interested in running our modified version of the Minecraft client, see below or click this link.

Just to keep things sane, we've got permissions set so that new users are in the DEFAULT group when they first join. That means you need to get a moderator or admin to put you in the PLAYERS group before you can access any commands..

Using Residence

We recently switched from Towny to Residence simply because it is easier. There are lots of tutorials on the web about how to use Residence. Like this one for example.
