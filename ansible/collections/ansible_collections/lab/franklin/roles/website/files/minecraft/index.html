How to Play

The server is running Forge 1.18.2 to make the server more fun.

  • Install the Curseforge Launcher
  • Log in and select Minecraft from the games list. Install Minecraft.
  • Download the server profile from Dropbox and import it
    • to the Curseforge Launcher.

The Server

From 2011 to 2016 the server ran CraftBukkit.
There are lots of interesting mines and buildings that have been worked on over the years. Also there is a rail system that can transport players around the map.
  • Our server address is minecraft.bitsmasher.net (159.203.166.26).
  • The server is not compatible with the Bedrock edition of Minecraft.
  • The server is running the Java edition of Minecraft..
