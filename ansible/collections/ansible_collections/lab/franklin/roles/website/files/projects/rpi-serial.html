Make connections

  • Connect GND pin from R-PI GPIO to GND pin on Pololu adapter.
  • Connect RX pin on the R-PI GPIO to the TX pin on Pololu adapter.
  • Connect TX pin on the R-PI GPIO to the RX pin on Pololu adapter.

Making console connections

Hardware Setup

USB to TTL Serial Cable - Debug / Console Cable f or Raspberry Pi
  • The red lead provides 5V of power at 500mA from the USB port to the Pi
  • The green lead transmits (TX) data from the USB port to the Pi
  • The white lead receives (RX) data into the USB port from the Pi
  • The black lead provides electrical ground to the system
Software Setup
  • In /boot/config.txt set enable_uart=1
  • Install ser2net like so sudo apt install ser2net -y
  • Edit the /etc/ser2net.yaml file.
  • Now run systemctl status ser2net
You can check the serial attachments with the command dmesg | grep ttyUSB
