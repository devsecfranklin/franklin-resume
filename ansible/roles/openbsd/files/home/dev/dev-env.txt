## Read flash

avrdude -U flash:r:/home/franklin/dev/optiboot_attiny841_8000000L_ser1_int.hex:i -P /dev/umodem0 -p attiny841 -c stk500v1 -C /home/franklin/dev/avrdude.conf

## Burn Bootloader

avrdude -p attiny841  -P /dev/umodem0 -c avrisp -b 19200 -U flash:w:release.hex

## Write Flash

-v -p attiny841 -c stk500v1
